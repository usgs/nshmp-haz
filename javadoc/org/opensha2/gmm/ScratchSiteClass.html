Interface ScratchSiteClass


  • public interface ScratchSiteClass
    Marker interface for any geotechnical site class heirarchy.

    A site class consists of discrete, named categories that correspond to ranges of Vs30 values. Implementations of this interface provide a Converter between scalar Vs30 values and specific site classes. Becasue each site class represents a range of Vs30, conversions are not invertible. For example, given a site class that spans [600..1200] m/s: 

     SiteClass.converter().convert(800).
 
 
 and between scalar Vs30 values and site class ScratchSiteClass.Intervals.
