      • main

        public static void main(String[] args)
        Entry point for the deaggregation of probabilisitic seismic hazard.

        Deaggregating siesmic hazard is largeley identical to a hazard calculation except that a return period (in years) must be supplied as an additional argument after the 'site(s)' argument. See the HazardCalc program for more information on required parameters.

        Please refer to the nshmp-haz wiki for comprehensive descriptions of source models, configuration files, site files, and hazard calculations.

        See Also:
        nshmp-haz wiki, example calculations

      • calc

        public static Deaggregation calc(Hazard hazard,
                 double returnPeriod)
        Deaggregate probabilistic seismic hazard at the supplied return period (in years). Deaggregation currently runs on a single thread.

        Call this method with the Hazard result of HazardCalc.calc(HazardModel, CalcConfig, Site, Optional) to which you supply the calculation settings and sites of interest that will also be used for deaggregation.

        Note: any model initialization settings in config will be ignored as the supplied model will already have been initialized.

        Parameters:
        returnPeriod - at which to deaggregate
        Returns:
        a Deaggregation object
