    public abstract class DataArray
extends AbstractList<Double>
    An immutable array of double values. This class is similar to the list returned by Doubles.asList(double...) in that it provides a thin List wrapper around a primitive double[] array, except all mutation operations throw an UnsupportedOperationException. This class provides getValue(int) for direct access to primitive value types to avoid autoboxing overhead.
